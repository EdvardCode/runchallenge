# Run Challenge 🏃

A mod for streamers and challenges in Minecraft! Create distance running challenges with progress tracking.

## 📋 Description

Run Challenge adds a challenge system with running for a set distance. The mod is perfect for streamers who want to create interactive challenges for themselves or their viewers.

## ✨ Features

- **🎯 Distance Challenges** - set any distance for the run
- **📊 BossBar** - visual progress bar with distance covered
- **↩️ Reverse Count** - counter decreases when moving backwards
- **🎆 Fireworks** - 5 volleys of colorful fireworks upon challenge completion
- **👥 Multiplayer** - support for multiple players, can manage other players' challenges
- **🖥️ Convenient GUI** - graphical interface for management (R key)
- **📝 Commands** - full control via commands
- **🌍 Localization** - Russian and English language

## 🎮 How to Use

### GUI (R key):
- Select a player from the dropdown list
- Specify the distance in blocks
- Press "Start" to begin the challenge
- Use buttons for management (teleport to start, push back)

### Commands:
- `/runchallenge start <distance> [player]` - start challenge
- `/runchallenge stop [player]` - stop challenge
- `/runchallenge status [player]` - show progress
- `/runchallenge tpstart [player]` - teleport to start
- `/runchallenge pushback [player]` - push back 10 blocks
- `/runchallenge players` - online players list

## 🎥 For Streamers

The mod is great for:
- Interactive streams with viewers
- Speed/endurance challenges
- Viewer punishments/rewards on a server
- Player competitions

## 📦 Installation

1. Install Minecraft Forge 1.20.1
2. Download the mod and place it in the `mods` folder
3. Launch the game

## 🔧 Requirements

- Minecraft 1.20.1
- Forge 47.3.0 or higher
